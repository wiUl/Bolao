# Bolão do Brasileirão

Sistema web para gerenciamento de bolão do Campeonato Brasileiro.

## Funcionalidades (MVP)
- Cadastro e login de usuários
- Cadastro de jogos e rodadas
- Envio de palpites
- Cálculo automático de pontuação
- Ranking dos participantes
